# MN4 跨脑图卡片匹配

MarginNote 4 插件，用于为题目脑图绑定答案脑图，选中题目卡片后快速查找并展示对应答案。

## 安装

在 MarginNote 4 插件页导入。

插件每 12 小时自动检查一次 [GitHub Releases](https://github.com/sinner7620/mn4_answer_matcher/releases)，发现正式版或测试版后可直接下载并交给 MarginNote 覆盖安装；也可从插件菜单选择“检查插件更新”。覆盖升级会保留原有脑图绑定记录，并在安装前额外写入恢复备份。

## 使用

| 渠道 | 当前版本 | 说明 |
| --- | --- | --- |
| 正式版 | [`v1.9.7`](https://github.com/sinner7620/mn4_answer_matcher/releases/tag/v1.9.7) | 稳定的跨脑图答案匹配 |
| Beta | [`v2.3.0-beta.1`](https://github.com/sinner7620/mn4_answer_matcher/releases/tag/v2.3.0-beta.1) | 答案匹配、正则规则匹配、章节顺序配对与错题管理 |

Beta 使用独立插件 ID，不会覆盖正式版。可以同时安装，但不建议同时启用，以免出现重复的卡片侧边按钮。

## 正式版功能

- 每个题目脑图绑定对应的答案脑图；
- 根据卡片标题和父节点路径匹配答案；
- 支持图片、评论、手写内容、合并摘录和子卡片；
- 答案窗口支持拖动和调整大小；

## Beta 功能

Beta 包含正式版的答案匹配功能，并新增：

- 可选择“完整标题匹配”“父节点标题匹配＋子卡片顺序”或“独立正则规则匹配”；
- 正则模式分别配置题目规则和答案规则，只按捕获组生成的匹配键查找，不进入其他匹配方式的优先级或回退链；
- 题目与答案正则规则仅在选择“独立正则规则匹配”后显示；
- 顺序模式只配对父标题唯一且两侧直接子卡片数量相同的章节，确认后固定保存卡片 ID；
- 数量不一致、父节点重名或没有唯一对应父节点时不自动顺延，未配对卡片继续回退到完整标题匹配；
- 错题工作台；
- 卡片侧边快速查答案和标记错题；
- 错题0级至错题5级：未掌握、已理解、可完成、已掌握、已稳定、已迁移；
- 错题搜索、分类筛选和跨脑图定位；
- 支持逐级选择父子节点分类；
- 错题总览、状态统计和来源分布；
- 来源分布按题目脑图根节点 ID 独立统计，同一学习集内的不同脑图不再合并为相同来源；
- 到期复习、添加时间和复习剩余天数；
- 到期列表直接支持跳转原题、查看实时答案、选择新状态并完成标记；

## 安装

1. 前往 [GitHub Releases](https://github.com/sinner7620/mn4_answer_matcher/releases)；
2. 下载正式版或 Beta 的 `.mnaddon` 文件；
3. 使用 MarginNote 4 打开并安装；
4. 完全关闭并重新打开 MarginNote。

## 使用

1. 打开题目脑图；
2. 在插件设置中绑定对应的答案脑图；
3. 选中题目卡片，点击侧边的“查找答案”；
4. Beta 用户可点击“标记错题”，随后在工作台中浏览和复习。

## 兼容性

- MarginNote 4.0.0 或更高版本；
- iPadOS 与 macOS。
